SecOT+ Flashcard Study Tool

A free, browser-based flashcard study tool designed to help learners prepare for the CompTIA SecOT+ certification exam.

Overview

This project provides interactive flashcards based on CompTIA's published SecOT+ exam objectives. The goal is to make exam preparation more efficient through active recall and repeated review of key concepts, terminology, and technologies relevant to operational technology (OT) and industrial cybersecurity.

The application runs entirely in the browser and requires no installation or account creation.

Features
